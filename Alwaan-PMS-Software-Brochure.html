Chapter 1

What the System Does

Alwaan PMS is a single unified platform that replaces spreadsheets, paper files, WhatsApp threads, and separate apps. It covers the entire lifecycle of a rental property — from the moment a tenant signs their Ejari to the day they move out.

End-to-End Coverage

The system handles every operational task in a property management company, including:

  • 📄 Digital tenant onboarding with Emirates ID scanning
  • 🏠 Unit inventory with occupancy tracking
  • 📋 Tenancy contract generation (DLD / Ejari format ready)
  • 💳 Post-dated cheque management with bank reconciliation
  • 📧 Automated email notifications for rent, renewals, violations
  • 🛠️ Maintenance ticket workflow between tenant → staff → vendor
  • 📊 Real-time financial dashboards for executives and owners
  • 📁 Secure document storage (EID, Ejari, cheques, passport)
  • 🔐 Role-based login for staff, owners, and tenants
  • 💰 Invoice generation, payment tracking, overdue alerts

The Problem It Solves

Most UAE property management companies rely on Excel files, paper ledgers, and disconnected tools. Information gets lost. Cheques get misplaced. Owners have to ask for updates by phone. Tenants can't find their invoices. Staff spend hours answering the same questions.

Alwaan PMS eliminates these pain points by giving every stakeholder — staff, owner, tenant — their own secure portal with real-time visibility into what matters to them.

Chapter 5

Security & Technology

The system is built on modern, battle-tested infrastructure — the same technology that powers global SaaS applications.

Security Measures

  • ✓
    SSL/TLS encryption — all data in transit is HTTPS-encrypted end-to-end
  • ✓
    Bcrypt password hashing — passwords are never stored in plain text
  • ✓
    HMAC-signed magic links — time-limited, tamper-proof login URLs
  • ✓
    HttpOnly session cookies — protection against XSS attacks
  • ✓
    Role-based data isolation — owners only see their building; tenants only see their unit
  • ✓
    Automated database backups — point-in-time recovery
  • ✓
    Every action logged — complete audit trail of who did what
  • ✓
    Cloud-hosted — 99.9% uptime guarantee
